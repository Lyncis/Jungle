## All together
### Installation
```bash
git clone https://github.com/Lyncis/Jungle.git
cd Jungle
npm install
cd backend
npm install
cd ../client
npm install
cd ../
```

### Run
```bash
npm start
```

## Separate backend and client running
```bash
# backend
cd backend
npm install
npm start

# client
cd client
npm install
npm start
```
